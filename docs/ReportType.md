# ReportType

The type of the report. - `Order`: Summary of all merchants' order. - `OrderTransaction`: Transaction summary of all merchants' orders. - `TopUpTransaction`: Transaction summary of all top ups. - `PayinWeeklyStatement`: Weekly report of all payins (including order-in and top up). - `PayinDailyStatement`: Daily report of all payins (including order-in and top up). - `CryptoPayout`: Transaction summary of all crypto settlements. - `OffRamp`: Transaction summary of all off ramp settlements. - `Refund`: Transaction summary of all refunds. - `PayoutWeeklyStatement`: Weekly report of all payouts (including crypto settlements, off ramp settlements and refund). - `PayoutDailyStatement`: Daily report of all payouts (including crypto settlements, off ramp settlements and refund). - `PayinCommissionFee`: Commission fee summary of all payins. - `PayoutCommissionFee`: Commission fee summary of all payouts. - `BalanceChange`: Balance change summary of all merchants. - `Summary`: Summary of all payins, payouts and commission fees.
