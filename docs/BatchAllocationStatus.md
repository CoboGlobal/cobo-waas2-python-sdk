# BatchAllocationStatus

The current status of the batch allocation. - `Pending`: The batch allocation has been created and is waiting to be processed. - `Processing`: The batch allocation is being processed. - `Completed`: All allocation items are completed. - `PartiallyCompleted`: Some allocation items are completed and some allocation items failed, with no allocation items still in progress. - `Failed`: All allocation items failed.
